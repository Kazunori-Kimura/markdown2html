Markdown to Html
---

Markdown形式のテキストファイルをHTMLに変換し、標準出力/ファイルへoutput

    >node markdown2html.js --help
    Usage: node D:\workspace\markdown2html\markdown2html.js [options] markdown
    Show this help message
        --help
    出力するHTMLのファイルパスを指定します。
        -o, --output-file
